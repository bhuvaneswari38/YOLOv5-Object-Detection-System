# Real-Time Object Detection System using YOLOv5

## 📌 Project Overview

This project focuses on developing a real-time object detection system using computer vision and deep learning techniques. The system captures live video input and detects multiple objects in real time using the YOLOv5 algorithm.

The implementation is designed to run on edge devices, enabling efficient and low-latency performance for real-world applications.

---

## ⚙️ Technologies Used

* Python
* OpenCV
* YOLOv5
* NVIDIA Jetson Nano 2GB
* Jetson Inference

---

## 🧠 System Implementation

* Captured real-time video using CSI IMX219 camera
* Processed frames using OpenCV
* Integrated YOLOv5 model for object detection
* Generated bounding boxes with class labels
* Optimized performance for edge AI deployment
